# Spam Email Classification
This project implements a machine learning solution to automatically filter unwanted "Spam" messages from legitimate "Ham" communications. By leveraging Natural Language Processing (NLP) techniques, the model identifies patterns and keywords common in fraudulent or unsolicited emails.
## Dataset
Dataset has two columns Category and Message. Category contains 2 label 'Spam' indicates that the email is classified as spam. 'Ham' denotes that the email is legitimate (ham). Message column contains the actual content of the email messages.
## Goal
The primary objective is to build a model that can:
* Pre-process raw text data.
* Classify incoming messages with high accuracy to minimize "False Positives" (legitimate emails being marked as spam).
